# ASHOK BHATTARAI — Portfolio

This repository contains a simple, responsive portfolio website for Ashok Bhattarai.

Files:
- `index.html` — the main portfolio page
- `styles.css` — stylesheet

How to preview locally:
1. Clone the repository:
   git clone https://github.com/your-username/portfol.git
2. Open `index.html` in your browser.

How to publish with GitHub Pages:
1. Push the files to the repository (main branch).
2. Go to the repository Settings → Pages.
3. Select branch `main` (or `gh-pages`) and root folder, then Save.
4. After a few minutes your site will be available at `https://your-username.github.io/portfol/`.
